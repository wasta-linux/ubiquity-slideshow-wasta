Functional and familiar

Regardless of what platform you're coming from, Wasta-Linux has an intuitive interface that is made to look familiar. The default desktop looks and feels a lot like Microsoft Windows 7/XP, while Wasta-Layout allows users to select interfaces inspired by macOS or Ubuntu Unity.

 

All of these options are made possible by using the Cinnamon desktop from the Linux Mint team. The Ubuntu base for Wasta-Linux ensures that users and developers alike benefit from the well established Ubuntu ecosystem.
